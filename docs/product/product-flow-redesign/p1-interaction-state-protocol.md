# P1：跨页交互与状态协议

日期：2026-07-10

结论：`PASS_WITH_LIMITS`

## 已落地协议

- Today 根据当前任务类型进入准备、知识、机会、面试或复盘工作台，不再要求用户自行寻找模块。
- 主导航固定为“今日 / 准备 / 机会 / 面试 / 复盘”；统计和数据设置保留为低频入口。
- 机会工作台把选中记录与编辑/比较模式写入 URL，Android 返回键按“键盘 → 编辑态 → 详情态”逐级退出。
- 学习笔记保存后可直接练题，口述证据保存后可直接复盘，复盘保存后可查看规则整理，形成连续出口。
- 表单保留真实的保存、取消、删除确认、错误反馈和读回；本地同步操作不伪造网络等待。
- Android 输入法打开时 WebView 使用 `adjustResize` 缩至可用高度并隐藏底栏，关闭后恢复；输入框和主操作保持在键盘上方。
- 兼容读取历史 `AI评分` 字段，但新界面只显示“规则自检”，避免把本地 rubric 伪装成 AI 能力。

## 验收证据

- 浏览器真实走通“学习笔记 → 单题练习 → 保存口述证据 → 去复盘 → 保存复盘 → 看整理”。
- React 路由、Today 深链、机会 URL 状态及各工作台保存/删除路径均有回归测试。
- OnePlus 8 Pro 上面试输入法状态为 `384 × 487px`，输入框底部约 `333.5px`，无横向溢出；返回键关闭键盘后恢复 `384 × 792px` 与底部导航。

- P5 已补正式签名 APK、远端 HTTPS 登录/session、服务端保存读回和 Android 杀进程重启证据；本轮仍未实现离线冲突合并或完整跨设备版本历史。
